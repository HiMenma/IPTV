# UI增强功能使用说明

## 新功能概览

本次更新为IPTV应用添加了5个实用功能，让您的使用体验更加流畅便捷。

## 1. 自定义播放列表名称 📝

### 如何使用
1. 点击右下角的 **+** 按钮
2. 在弹出的对话框中，您会看到 **"播放列表名称（可选）"** 输入框
3. 输入您喜欢的名称，例如："我的直播源"、"体育频道"等
4. 如果不输入名称，系统会自动使用默认名称

### 小贴士
- 支持中文输入
- 名称会显示在播放列表中，方便识别
- 可以随时通过编辑按钮修改名称

## 2. 加载进度提示 ⏳

### 功能说明
添加播放列表时，您会看到一个加载对话框，显示当前操作状态：
- **M3U播放列表**: "正在加载播放列表..."
- **Xtream服务器**: "正在连接Xtream服务器..."

### 好处
- 清楚了解操作进度
- 避免重复点击
- 出错时会显示详细的错误信息

## 3. 视频加载动画 🎬

### 功能说明
打开直播频道时，在视频加载期间会显示：
- 半透明的黑色背景
- 旋转的加载动画
- "正在加载..." 提示文字

### 好处
- 知道视频正在加载中
- 不会误以为应用卡住了
- 视频开始播放后自动消失

## 4. 刷新播放列表 🔄

### 如何使用
1. 在播放列表界面，找到您想刷新的播放列表
2. 点击播放列表右侧的 **刷新按钮**（圆形箭头图标）
3. 等待刷新完成

### 注意事项
- ✅ **M3U URL播放列表**: 完全支持刷新
- ⚠️ **Xtream播放列表**: 暂不支持刷新（需要重新添加）
- ❌ **本地M3U文件**: 不显示刷新按钮（无法刷新）

### 使用场景
- 播放列表更新了新频道
- 某些频道链接失效需要更新
- 定期刷新获取最新内容

## 5. 全屏播放 📺

### 如何使用
1. 打开任意直播频道
2. 点击右上角的 **"全屏"** 按钮
3. 视频会占满整个屏幕
4. 点击 **"退出全屏"** 按钮返回正常模式

### 特点
- 全屏时顶部导航栏自动隐藏
- 视频占满整个屏幕
- 提供更好的观看体验
- 随时可以退出全屏

## 常见问题

### Q: 为什么Xtream播放列表不能刷新？
**A**: 当前版本暂不支持Xtream刷新功能，因为需要存储账户密码。如需更新，请删除后重新添加播放列表。

### Q: 加载对话框一直显示怎么办？
**A**: 可能是网络问题或URL无效。请检查：
- 网络连接是否正常
- URL是否正确
- 服务器是否在线

### Q: 全屏按钮在哪里？
**A**: 在播放器界面的右上角，显示为半透明的按钮，文字为"全屏"或"退出全屏"。

### Q: 如何取消正在加载的播放列表？
**A**: 当前版本暂不支持取消操作，请等待加载完成或失败。

## 使用技巧

### 技巧1: 合理命名播放列表
给播放列表起一个有意义的名称，例如：
- "央视频道"
- "地方台"
- "体育直播"
- "电影频道"

这样可以快速找到想看的内容。

### 技巧2: 定期刷新播放列表
建议每周刷新一次M3U播放列表，确保频道链接是最新的。

### 技巧3: 全屏观看体验更佳
观看直播时使用全屏模式，可以获得更好的沉浸式体验。

### 技巧4: 注意加载提示
如果加载时间过长，可能是：
- 网络速度慢
- 服务器响应慢
- URL有问题

可以尝试更换网络或URL。

## 快速操作指南

### 添加新播放列表
```
1. 点击 + 按钮
2. 输入名称（可选）
3. 输入URL或服务器信息
4. 点击"添加"
5. 等待加载完成
```

### 刷新现有播放列表
```
1. 找到要刷新的播放列表
2. 点击刷新按钮
3. 等待刷新完成
```

### 全屏观看
```
1. 打开频道
2. 点击"全屏"按钮
3. 享受观看
4. 点击"退出全屏"返回
```
