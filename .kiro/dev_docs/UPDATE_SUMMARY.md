- ❌ 移除了主界面AppBar右上角的收藏和历史按钮

### 使用底部导航栏
- ✅ 底部导航栏有三个标签：
  - 🏠 **Home** - 配置列表
  - ❤️ **Favorites** - 收藏的频道
  - 🕐 **History** - 播放历史

### 改进功能
- ✅ **自动刷新**: 切换标签时自动加载最新数据
- ✅ **手动刷新**: 每个页面AppBar有刷新按钮
- ✅ **状态保持**: 页面状态和滚动位置保持

## 使用方法

### 1. 添加收藏
```
Home标签 → 选择配置 → 频道列表 → 点击❤️图标
```

### 2. 查看收藏
```
点击底部 ❤️ Favorites 标签 → 看到所有收藏的频道
```

### 3. 查看历史
```
播放频道 → 返回 → 点击底部 🕐 History 标签 → 看到播放历史
```
